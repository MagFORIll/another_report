# CTR Analyzer

Приложение для обработки файлов и расчета данных с выводом в табличном виде.

## Быстрый старт

Склонируйте репозиторий:

```bash
git clone https://github.com/MagFORIll/another_report
cd another_report
```

Установите зависимости:

```bash
pip install -r requests.txt
```

Если в `requests.txt` отсутствуют необходимые зависимости, сообщите об этом мне.

Запуск приложения:

```bash
python main.py [--files <файлы через пробел>] [--report <имя отчета>]
```

## Функциональность

* Обработка файлов из папки `docs`
* Подсчет данных и формирование итоговой таблицы
* Генерация отчета

### Аргументы командной строки

* `--files` список файлов для обработки через пробел
* `--report` имя выходного файла

### Поведение по умолчанию

* Используются все файлы из папки `docs`
* Имя отчета: `average-ctr`

Важно: если файлы находятся не в папке `docs`, они не будут обработаны.

## Структура проекта

```
project/
├── handler.py      # Подготовка приложения, поиск файлов, расчет данных
├── parser.py       # Обработка аргументов командной строки (--files, --report)
├── input_file.py   # Чтение данных из файлов
├── output_file.py  # Создание файла с результатами
├── main.py         # Точка входа, объединяет весь функционал
└── docs/           # Папка с входными файлами
```

## Архитектура

Логика проекта разделена на модули, чтобы упростить поддержку, облегчить добавление нового функционала и сделать код более читаемым.

## Пример использования

```bash
python main.py --files stats1.csv stats2.csv --report result
```
